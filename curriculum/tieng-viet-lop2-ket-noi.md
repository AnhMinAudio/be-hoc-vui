# Khung chuẩn — TIẾNG VIỆT LỚP 2 (Kết nối tri thức với cuộc sống)

> Khung ra đề (phần kiến thức KHÁCH QUAN: Luyện từ và câu, chính tả) bám chương trình GDPT 2018.
> Nguồn: SGK/giáo án Tiếng Việt 2 KNTT (lop2.vn, vietjack, vndoc); Chương trình GDPT 2018 môn Tiếng Việt.
> Cập nhật: 2026-05-28. (Lưu ý: phần Đọc hiểu/Viết đoạn mang tính chủ quan → ưu tiên ra phần Luyện từ và câu để chấm khách quan.)

| Mã | Yêu cầu cần đạt |
|---|---|
| TV2-TU-1 | Nhận biết từ chỉ SỰ VẬT (người, vật, cây cối, con vật, hiện tượng) |
| TV2-TU-2 | Nhận biết từ chỉ HOẠT ĐỘNG |
| TV2-TU-3 | Nhận biết từ chỉ ĐẶC ĐIỂM (màu sắc, hình dáng, tính tình…) |
| TV2-CAU-1 | Câu giới thiệu kiểu "Ai là gì?" |
| TV2-CAU-2 | Câu kiểu "Ai làm gì?" |
| TV2-CAU-3 | Câu kiểu "Ai thế nào?" |
| TV2-CAU-4 | Đặt và trả lời câu hỏi: Ai? Cái gì? Làm gì? Khi nào? Ở đâu? Vì sao? Để làm gì? |
| TV2-DAU-1 | Dấu chấm (cuối câu kể) |
| TV2-DAU-2 | Dấu chấm hỏi (cuối câu hỏi) |
| TV2-DAU-3 | Dấu chấm than (câu cảm, câu khiến) |
| TV2-DAU-4 | Dấu phẩy (ngăn cách các từ cùng loại) |
| TV2-VT-1 | Mở rộng vốn từ theo chủ đề: bốn mùa, con vật, cây cối, nghề nghiệp, gia đình, trường học, đồ vật |
| TV2-NGHIA-1 | Từ trái nghĩa (cao–thấp, to–nhỏ, sáng–tối…) |
| TV2-NGHIA-2 | Từ gần nghĩa / nghĩa của từ đơn giản |
| TV2-CT-1 | Chính tả: phân biệt c/k, g/gh, ng/ngh, l/n, s/x, dấu hỏi/ngã (mức cơ bản) |

**Loại câu phù hợp:** trắc nghiệm, điền đáp án (có alternatives), đúng/sai, ghép cặp (cột phải khác nhau).
